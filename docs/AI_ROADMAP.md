# AI Upgrade Roadmap

이 프로젝트를 AI 개발자 포트폴리오로 키우기 위한 단계별 계획입니다.

## 1단계: 현재 버전 정리

목표: 설치 없이 실행되는 사진 합성 앱 완성

- HTML/Canvas 기반 UI
- 이미지 업로드
- 자동 맞춤 휴리스틱
- 장식 룰렛 애니메이션
- 결과 이미지 저장

## 2단계: 브라우저 모델 붙이기

목표: 강아지 전체 영역을 모델로 찾기

- TensorFlow.js COCO-SSD 모델 적용
- `dog` bounding box 탐지
- 탐지된 박스의 상단 중심을 기준으로 귀/모자 위치 자동 계산
- 모델 로딩 실패 또는 탐지 실패 시 기존 자동 맞춤으로 fallback

현재 구현 상태: 완료

좋은 포트폴리오 포인트:

- 모델 로딩 UX
- 추론 시간 측정
- 모델 실패 시 기존 자동 맞춤으로 fallback

## 3단계: 릴스형 사용자 인터랙션

목표: 릴스형 합성 UX와 더 정확한 얼굴 기준점을 연결하기

- 사용자가 직접 멈추는 릴스형 인터랙션 구현
- 얼굴 기준점에 가까우면 장식이 스냅되는 로직 구현
- 사진에서 귀처럼 보이는 영역을 잘라 `사진 귀 컷` 장식으로 사용

현재 구현 상태: 완료

## 4단계: 커스텀 강아지 얼굴 모델

목표: 강아지 얼굴, 귀, 머리 위치를 더 정확하게 찾기

- 강아지 얼굴 데이터셋 수집
- 라벨링: 얼굴 박스, 왼쪽 귀, 오른쪽 귀, 머리 중심
- 학습: object detection 또는 keypoint detection
- 브라우저용 모델 변환 또는 작은 백엔드 API 구성

좋은 포트폴리오 포인트:

- 데이터 라벨링 기준 문서화
- 학습/검증 지표 기록
- 모델 전후 비교 이미지
- 실패 케이스 분석

## 추천 순서

바로 무거운 모델부터 시작하기보다, 먼저 GitHub에 현재 앱을 올리고 2단계 브라우저 모델을 붙이는 것을 추천합니다. 그래야 앱, Git, 모델 추론, UX를 한 번에 연결한 결과물이 생깁니다.
